### Description:

Given a string s, return the longest palindromic substring in s.

#### Example 1:
```swift
Input: s = "babad"
Output: "bab"
Note: "aba" is also a valid answer.
```

#### Example 2:
```swift
Input: s = "cbbd"
Output: "bb"
```

#### Example 3:
```swift
Input: s = "a"
Output: "a"
```

#### Example 4:
```swift
Input: s = "ac"
Output: "a"
```

#### Constraints:
```swift
1 <= s.length <= 1000
s consist of only digits and English letters (lower-case and/or upper-case)
```
